# Myntra Inventory Data Analysis (Excel)

## Project Overview
This project focuses on analyzing Myntra inventory data using Microsoft Excel.
The dataset was cleaned, processed, and visualized to create an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

## Key Work Done
- Removed duplicates and handled missing values
- Cleaned raw inventory data
- Created pivot tables for analysis
- Designed an interactive Excel dashboard using slicers

## Dataset
- Myntra Inventory Dataset (cleaned)
